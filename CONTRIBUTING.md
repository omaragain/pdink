# Contributing to Pdink

Thanks for helping improve Pdink.

## Before opening an issue

- Check whether the issue already exists.
- Include your Windows version, Pdink version, input file type, selected OCR
  language(s), and the exact error text.
- Do not upload confidential documents. Use a redacted sample when possible.

## Pull requests

- Keep changes focused and explain what they solve.
- Preserve local-first behavior: source files must not be uploaded during
  conversion.
- Test Markdown and plain-text output before opening a pull request.
- Do not add more than three OCR languages to a single conversion job.
