# Changelog

All notable changes to Pdink are documented in this file.

## [1.0.0] - 2026-07-05

### Added

- Local-first conversion to Markdown and plain text.
- Offline OCR for PDFs and images through Tesseract.
- OCR language selection with a strict one-to-three language limit.
- Bundled language-pack support.
- Persistent output location, output format, input folder, and language settings.
- Drag-and-drop file queue with background conversion.
- Windows icon, GitHub project link, and installer configuration.
